## 1. 概要

### 背景

[企画書 v0.3](../企画書.md) Phase 6。F-04（未購入に戻す）・F-08（削除）と購入履歴の一覧表示を実装する。
ストアの `togglePurchased` / `deleteItem` は実装済み。本 Phase は **新規ルート + UI** の追加のみ。

### 目的

- `/history` に購入済みアイテムの履歴画面を追加する
- 購入日ごとにグルーピング（今日 / 昨日 / それ以前は日付）
- 各行に「未購入に戻す（↶）」と「削除（🗑）」のボタンを設置
- メイン画面ヘッダーから履歴画面へ遷移できるアイコンボタンを追加

### 方針

| 観点 | 判断 |
|------|------|
| 画面 | Next.js App Router の独立ルート `/history` |
| 戻る | ヘッダー左の `←` でメイン (`/`) に遷移（Next.js `Link`） |
| グルーピング | `purchasedAt` の日付（ローカルタイム） で「今日」「昨日」「それ以前は日付（YYYY/MM/DD）」で分類 |
| 各グループ内ソート | `purchasedAt` 降順（新しい順） |
| 復元（未購入に戻す） | `togglePurchased(id)` 一発でスコープを保ったまま PENDING に戻る |
| 削除 | `deleteItem(id)`（確認モーダルなし、誤タップ救済は復元で代替できる) |
| 空状態 | 購入済みアイテムが0件の場合「購入済みアイテムはありません」表示 |
| 日付計算 | 標準 `Date` API を使用（外部ライブラリ追加なし） |

### 5-3. 機能テスト項目
