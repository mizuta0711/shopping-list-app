## 1. 概要

### 背景

[企画書 v0.3](../企画書.md) Phase 4。F-05（区分タブ）と F-06（区分間移動）を実装する。
データモデルとストアアクション（`scope`, `moveScope`）は Phase 1 で実装済み。本 Phase は **UI のみ**。

### 目的

- メイン画面に **「今日」/「また今度」のタブ** を追加し、未購入アイテムを区分ごとに表示する
- 各行に **区分切替ボタン** を追加し、`今日 ⇔ また今度` を 1 タップで移動できるようにする
- アイテム追加時、**現在アクティブなタブの区分** に追加する

### 方針

| 観点 | 判断 |
|------|------|
| タブ状態の管理 | コンポーネントローカル `useState`（URL ルーティングは MVP では不要） |
| タブのデフォルト | `TODAY` |
| アイテム件数表示 | 各タブに `今日(5) / また今度(3)` のように pending 件数を表示 |
| 区分切替ボタン | 行右端に 1 タップのアイコンボタン。`今日` 表示中は「→ また今度」、`また今度` 表示中は「→ 今日」を表示 |
| 切替後の挙動 | 即時にリストから消える（タブが切り替わったわけではないので、移動先タブには表示される） |
| 追加先 scope | アクティブなタブの scope。AddItemForm に props で渡す |
| アクセシビリティ | タブは `role="tablist"` / `role="tab"` を使用、キーボード矢印キー対応は MVP 対象外 |

### 4-2. コンポーネント API

#### `ScopeTabs`

```typescript
type Props = {
  active: ItemScope;
  todayCount: number;
  laterCount: number;
  onChange: (scope: ItemScope) => void;
};
```

- 横並び 2 つのタブボタン
- アクティブなタブ: 下線 + 太字
- 件数: 半透明の括弧書き

#### `ShoppingItemRow`（変更後）

```typescript
type Props = {
  item: ShoppingItem;
  onToggle: (id: string) => void;
  onMoveScope: (id: string, targetScope: ItemScope) => void;
};
```

- 行のメイン領域（チェック + 名前）は `togglePurchased` 用ボタン
- 行の右端に小さなアイコンボタンで `moveScope` を実行
- `onClick` のバブリングを止める（行全体タップとの競合回避）

#### `AddItemForm`（変更後）

```typescript
type Props = {
  scope: ItemScope;
};
```

- `addItem(value, scope)` を呼ぶ

### 4-3. レイアウト

```
┌─────────────────────────────┐
│ 🛒 買い物リスト              │
├─────────────────────────────┤
│  今日(3)    また今度(2)      │ ← タブ
│  ━━━━━━━                     │
├─────────────────────────────┤
│  ☐ 牛乳                  ⇨  │ ← ⇨ で「また今度」へ移動
│  ☐ 卵                    ⇨  │
│  ☐ パン                  ⇨  │
├─────────────────────────────┤
│ [追加したい商品名…]      [+] │
└─────────────────────────────┘
```

タブをタップすると上下のリストが切り替わる。`また今度` タブでは矢印アイコンが反対方向（⇦ 今日へ）になる。

## 5. ブラウザ評価計画

### 5-3. 機能テスト項目

| # | 操作手順 | 期待結果 |
|---|---------|---------|
| 1 | 空状態でアプリを開く | 「今日(0) / また今度(0)」のタブと空状態が表示される |
| 2 | 「今日」タブで「牛乳」を追加 | 「今日(1)」のリストに牛乳が表示される |
| 3 | 「また今度」タブをタップ | 「また今度(0)」の空状態が表示される |
| 4 | 「また今度」タブで「醤油」を追加 | 「また今度(1)」のリストに醤油が表示される |
| 5 | 「今日」タブをタップ | 牛乳が表示され、醤油は表示されない |
| 6 | 「牛乳」の右側ボタンをタップ | 牛乳が「今日」リストから消え、件数が「今日(0) / また今度(2)」になる |
| 7 | 「また今度」タブに切り替え | 「醤油」「牛乳」が表示される |
