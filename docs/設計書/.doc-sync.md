# 設計書同期トラッキング

コミットごとに設計書の同期状態を記録する。
`/update-docs` 実行時に自動追記される。プッシュ前チェックは `grep` でコミットハッシュの存在を確認するのみ。

## ルール

- `/update-docs` 実行時: 対象コミットのハッシュ・同期先・判定を追記
- 設計書更新不要なコミット（バグ修正等）: `更新不要` として記録
- プッシュ前チェック: `git log --format=%h @{upstream}..HEAD` の全ハッシュがこのファイルに存在するか `grep` で確認
