# API設計

## 共通仕様

- **認証**: NextAuth（認証不要エンドポイントを除き全て認証必須）
- **バリデーション**: Zod
- **レスポンス形式**:
  ```json
  { "success": true, "data": { ... } }
  { "success": false, "error": { "code": "ERROR_CODE", "message": "...", "fields": { "email": "..." } } }
  ```
- **ページネーション**: カーソルベース（`cursor`, `limit`パラメータ）
