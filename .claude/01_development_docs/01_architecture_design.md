## 5層レイヤードアーキテクチャ

```
┌─────────────────────────────────┐
│  Presentation Layer             │ React Components（page.tsx, feature components）
├─────────────────────────────────┤
│  State Layer                    │ Zustand Stores（APIキャッシュ + UI状態）
├─────────────────────────────────┤
│  Service Layer                  │ ビジネスロジック + AI Services
├─────────────────────────────────┤
│  Repository Layer               │ AI Provider抽象化（IAIRepository）
├─────────────────────────────────┤
│  Infrastructure Layer           │ PostgreSQL（Prisma） + External AI APIs
└─────────────────────────────────┘
```
