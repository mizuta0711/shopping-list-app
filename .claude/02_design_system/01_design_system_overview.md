# デザインシステム概要

## 技術スタック

- **Tailwind CSS v4** — ユーティリティファーストのスタイリング
- **スマホ専用**（375px 基準）。PC/タブレット幅のレイアウト最適化は対応不要

## 対応端末

| 端末 | 幅 | 対応方針 |
|---|---|---|
| スマートフォン | 375px 基準（〜428px 程度を許容） | フル対応 |
| タブレット・PC | 768px〜 | **対応不要**（スマホ幅に最適化されたレイアウトをそのまま中央配置で表示する想定） |

実装ガイドライン:
- `max-w-md` 程度でコンテンツ幅を制限し、PC で開いた場合も中央寄せで破綻させない
- Tailwind の `sm:` / `md:` / `lg:` ブレークポイントは原則使わない
- ナビゲーションは下部固定（タブバー）など、スマホネイティブUIを意識した配置を採用
