# ビルド＋品質チェック

以下を順番に実行し、結果をまとめて報告してください。

## 実行コマンド

### 1. TypeScript + ビルド
```bash
npm run build
```

### 2. Lint
```bash
npm run lint
```

## 結果報告

```
## ビルド＋品質チェック結果

### ビルド: ✅ 成功 / ❌ 失敗
（エラーがある場合は内容を記載）

### Lint: ✅ エラーなし / ⚠️ 警告あり / ❌ エラーあり
- エラー: N件
- 警告: N件（既存 / 新規を区別）
（エラー・警告の内容を記載）

### 総合判定: ✅ コミット可 / ❌ 修正必要
```

## エラーがある場合

ビルドエラーやlintエラーがある場合は、修正を提案してください。
ただし、既存のwarning（今回の変更に無関係なもの）は無視して構いません。
